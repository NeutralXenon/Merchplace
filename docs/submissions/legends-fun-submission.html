Short Description

Merchplace is a Solana-native marketplace for limited event merch: Breakpoint shirts, Colosseum winner gear, Superteam contributor drops, badges, caps, bags, and other collector artifacts. Sellers publish wallet-signed listings, buyers deposit USDC into escrow, and funds release only after the buyer confirms receipt.

Full Description

Solana events create scarce physical artifacts, but the resale flow is still mostly DMs, Discord trust, and manual payments. That works for friends, but it breaks down for collectors who want provenance, shipping clarity, and payment protection before buying a limited shirt, cap, badge, bag, or team drop. Merchplace is a marketplace for Solana event merch. Sellers create listings with item details, event context, images, condition, price, and a fixed carrier shipping method. Buyers see the full USDC total before signing: item price, shipping, and a 5% protection/platform fee. When they buy, USDC moves into a program-controlled escrow vault instead of going directly to the seller. The Solana part is not decorative. The Anchor program handles listing creation, escrow funding, buyer cancellation, receipt confirmation, seller proceeds, and treasury fee release. Off-chain Supabase metadata stores the rich marketplace content, while the on-chain listing stores the metadata hash, seller, buyer, price, shipping cost, mint, and status. Backend APIs verify confirmed on-chain state before accepting public listing or status updates. The current MVP includes browse, sell, listing detail, profile, wallet auth, image upload, event drop pages, shipment tracking, escrow guidance, dispute/timeout copy, and localnet lifecycle smoke tests for create, buy, receipt confirmation, buyer cancellation, and seller cancellation. The launch wedge is event-specific shelves, starting with Breakpoint and Colosseum inventory. Merchplace is already deployed and smoke-tested on devnet, with a full generated-wallet lifecycle covering create listing, buy, shipment tracking, and receipt confirmation. Next, it needs a hosted app URL, a public GitHub repo, and real seller seed inventory. The goal is to turn Solana event merch into a trustable, collectible marketplace instead of a scattered chat-channel trade.

Why Solana

Merchplace uses Solana for the trust boundary that matters: custody and settlement. Buyers can verify that their USDC is held in escrow, sellers can see a wallet-signed listing and status trail, and the marketplace can sync off-chain metadata only after confirmed on-chain state. Fast, low-cost transactions make small physical-item sales practical without hiding payment state behind a centralized marketplace ledger.
